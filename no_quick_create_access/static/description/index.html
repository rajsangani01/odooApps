Relational Create Restrict

Disable Create & Create/Edit Options in Relational Fields (Odoo 19 Community)

Overview

This module allows administrators to disable the Create and Create & Edit options in relational fields (Many2one, Many2many, etc.). You can apply restrictions globally per user or model-wise for selected models.

Key Features

  • Hide Create and Create & Edit from relational field dropdowns
  • Global restriction per user
  • Model-specific restriction settings
  • Admin users bypass restrictions
  • User-friendly configuration interface
  • Reset button to clear all restrictions
  • Works with all relational fields in Odoo 19

Configuration

Navigate to:
Settings → Users → Relational Restrictions

  • Hide Create / Create & Edit — disable creation globally
  • Model-wise Restriction — select models to restrict
  • Reset Restrictions — clear all values in one click
