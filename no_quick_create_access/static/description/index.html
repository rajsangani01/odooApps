Relational Create Restrict

Disable Create & Create and Edit in Many2one / Many2many Fields

This module allows administrators to restrict the creation of records from relational fields such as Many2one and Many2many. You can apply a global restriction for a user or restrict only specific models as needed.

Features

  • Hide Create and Create & Edit options in relational dropdowns
  • Global restriction per user
  • Model-wise restriction for selected models
  • User-friendly relational restriction settings
  • Admin bypass (admin users are never restricted)
  • Includes Reset Restrictions button for quick cleanup
  • Compatible with all relational fields in Odoo 19

Configuration

Go to:
Settings → Users → Relational Restrictions

  • Hide Create / Create & Edit — disable creation for all relational fields
  • Model-wise Restriction — restrict creation only for selected models
  • Reset Restrictions Button — clear all restrictions for the selected user
